Como Instalar e Configurar NGINX

15 minutos
Iniciante
Servidor Web
Home / Tutoriais / NGINX

1 Atualizando o Sistema

Primeiro, vamos atualizar a lista de pacotes e o sistema:

bash
sudo apt update && sudo apt upgrade -y

2 Instalando o NGINX

Instale o NGINX usando o gerenciador de pacotes apt:

bash
sudo apt install nginx -y
Instalação Completa
O NGINX foi instalado com sucesso no seu sistema.

3 Iniciando e Habilitando o NGINX

Inicie o serviço e configure para iniciar automaticamente:

bash
sudo systemctl start nginx sudo systemctl enable nginx

Verifique o status do serviço:

bash
sudo systemctl status nginx

4 Configurando o Firewall

Configure o firewall para permitir tráfego HTTP e HTTPS:

bash
sudo ufw allow 'Nginx Full' sudo ufw status
Atenção
Certifique-se de que o UFW está habilitado antes de configurar as regras do firewall.

5 Testando a Instalação

Acesse seu servidor no navegador para verificar se o NGINX está funcionando:

url
http://localhost # ou http://seu-ip-do-servidor

Você deve ver a página padrão do NGINX com "Welcome to nginx!"

6 Configuração Básica

Edite o arquivo de configuração principal:

bash
sudo nano /etc/nginx/sites-available/default

Exemplo de configuração básica para um site:

nginx
server { listen 80; server_name exemplo.com www.exemplo.com; root /var/www/html; index index.html index.php; location / { try_files $uri $uri/ =404; } location ~ \.php$ { include snippets/fastcgi-php.conf; fastcgi_pass unix:/var/run/php/php8.1-fpm.sock; } }

Teste a configuração e recarregue o NGINX:

bash
sudo nginx -t sudo systemctl reload nginx

1 Download do NGINX

Baixe a versão mais recente do NGINX para Windows:

url
https://nginx.org/en/download.html

Baixe o arquivo nginx/Windows-1.24.0 (versão estável)

2 Extraindo e Configurando

Extraia o arquivo ZIP para uma pasta (exemplo: C:\nginx):

cmd
C:\nginx\ ├── conf/ ├── html/ ├── logs/ └── nginx.exe

3 Iniciando o NGINX

Abra o Prompt de Comando como Administrador e navegue para a pasta do NGINX:

cmd
cd C:\nginx nginx.exe

Para parar o NGINX:

cmd
nginx.exe -s stop

4 Configuração como Serviço

Para executar o NGINX como serviço do Windows, use o NSSM:

url
https://nssm.cc/download

Instale o serviço:

cmd
nssm install nginx C:\nginx\nginx.exe nssm start nginx

5 Testando a Instalação

Acesse no navegador:

url
http://localhost
Instalação Completa
O NGINX está funcionando no Windows!

1 Instalando o Homebrew

Se você não tem o Homebrew instalado:

bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

2 Instalando o NGINX

Use o Homebrew para instalar o NGINX:

bash
brew install nginx

3 Iniciando o NGINX

Inicie o serviço do NGINX:

bash
brew services start nginx

Para parar o serviço:

bash
brew services stop nginx

4 Configuração

Os arquivos de configuração estão em:

bash
nginx config: /usr/local/etc/nginx/nginx.conf documentroot: /usr/local/var/www

Edite a configuração:

bash
nano /usr/local/etc/nginx/nginx.conf

5 Testando a Instalação

Acesse no navegador:

url
http://localhost:8080
Instalação Completa
O NGINX está funcionando no macOS!
