Como Instalar e Configurar Laravel

25 minutos
Intermediário
Framework PHP
Home / Tutoriais / Laravel

1 Verificando Pré-requisitos

Antes de instalar o Laravel, certifique-se de que tem os requisitos:

bash
php -v composer --version
Requisitos Mínimos
PHP >= 8.1, Composer, extensões: mbstring, xml, zip, curl, mysql/pgsql

2 Instalando o Laravel Installer

Instale o Laravel Installer globalmente via Composer:

bash
composer global require laravel/installer

Adicione o diretório do Composer ao PATH:

bash
echo 'export PATH="$HOME/.composer/vendor/bin:$PATH"' >> ~/.bashrc source ~/.bashrc

3 Criando um Novo Projeto Laravel

Crie um novo projeto Laravel:

bash
laravel new meu-projeto # ou composer create-project laravel/laravel meu-projeto

Navegue para o diretório do projeto:

bash
cd meu-projeto

4 Configuração Inicial

Copie o arquivo de configuração de ambiente:

bash
cp .env.example .env

Gere a chave da aplicação:

bash
php artisan key:generate

Edite o arquivo .env para configurar o banco de dados:

bash
nano .env
env
DB_CONNECTION=mysql DB_HOST=127.0.0.1 DB_PORT=3306 DB_DATABASE=nome_do_banco DB_USERNAME=usuario DB_PASSWORD=senha

5 Instalando Dependências e Executando Migrações

Instale as dependências do Node.js:

bash
npm install

Execute as migrações do banco de dados:

bash
php artisan migrate
Atenção
Certifique-se de que o banco de dados está criado e as credenciais estão corretas no arquivo .env

6 Configurando Permissões

Configure as permissões corretas para as pastas de storage e cache:

bash
sudo chown -R www-data:www-data storage bootstrap/cache sudo chmod -R 775 storage bootstrap/cache

7 Iniciando o Servidor de Desenvolvimento

Inicie o servidor de desenvolvimento do Laravel:

bash
php artisan serve

Acesse no navegador:

url
http://localhost:8000
Laravel Instalado!
Você deve ver a página de boas-vindas do Laravel.

8 Comandos Essenciais do Artisan

Comandos úteis para desenvolvimento:

bash
# Criar um controller php artisan make:controller NomeController # Criar um model php artisan make:model NomeModel # Criar uma migration php artisan make:migration criar_tabela_exemplo # Limpar cache php artisan cache:clear php artisan config:clear php artisan view:clear # Listar todas as rotas php artisan route:list
